## About the Project
This project aims to learn an unknown probability density function (PDF) of a transformed random variable using a **Generative Adversarial Network (GAN)**. 

The core objective is to estimate the distribution of transformed Air Quality data ($NO_2$ concentrations) without assuming any parametric form (like Gaussian or Exponential), relying solely on the data samples generated by the GAN.

### Key Objectives
1. **Data Transformation:** Transform raw $NO_2$ data using a non-linear function dependent on a specific University Roll Number.
2. **GAN Training:** Train a Generator to mimic this transformed distribution and a Discriminator to distinguish between real and fake samples.
3. **PDF Estimation:** Use Kernel Density Estimation (KDE) to visualize and compare the real vs. generated probability distributions.

## Methodology

### 1. Data Transformation
The input feature $x$ (NO2) is transformed into $z$ using the following equation:

$$z = x + a_r \sin(b_r \cdot x)$$

Where parameters $a_r$ and $b_r$ are derived from the University Roll Number:
* $a_r = 0.5 \times (Roll \pmod 7)$
* $b_r = 0.3 \times ((Roll \pmod 5) + 1)$

### 2. GAN Architecture
* **Generator:** A Feed-Forward Neural Network that takes a noise vector (dim=5) and outputs a single value $z$.
  * *Structure:* Linear(5→32) → LeakyReLU → Linear(32→64) → LeakyReLU → Linear(64→1).
* **Discriminator:** A Binary Classifier that estimates the probability of a sample being real.
  * *Structure:* Linear(1→64) → LeakyReLU → Linear(64→32) → LeakyReLU → Linear(32→1) → Sigmoid.
